Языки программирования курса Колесника Кирилла

В рамках данной лабораторной работы я разработал набор HTML-документов, описывающих языки программирования, которые упоминаются в данном курсе.

Для оформления страниц использовались CSS-фреймворк Bootstrap и готовые компоненты библиотеки без написания собственных CSS-стилей.

На сайте реализованы стартовая страница, страница со списком языков программирования и отдельные страницы для каждого языка с описанием и примером кода.
