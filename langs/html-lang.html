Язык программирования HTML

Назад

Краткое описание

HTML (HyperText Markup Language) — это язык разметки, который используется для создания структуры веб-страниц в браузере.

С помощью HTML задаются заголовки, абзацы, списки, ссылки, изображения и другие элементы интерфейса, а внешний вид настраивается с помощью CSS.

  • Определяет логическую структуру документа.
  • Работает совместно с CSS и JavaScript.
  • Поддерживается всеми современными веб-браузерами.

Пример исходного кода

<!doctype html>
            <html lang="ru">
            <head>
                <meta charset="utf-8">
                <title>Пример HTML-страницы</title>
            </head>
            <body>
                <h1>Заголовок страницы</h1>
                <p>Это пример абзаца текста на HTML-странице.</p>
            </body>
            </html>
