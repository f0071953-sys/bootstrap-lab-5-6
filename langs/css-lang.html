Язык программирования CSS

Назад к списку Назад

Краткое описание

CSS (Cascading Style Sheets) — это язык таблиц стилей, который используется для оформления HTML-документов и управления внешним видом веб-страниц.

С помощью CSS можно задавать цвета, шрифты, отступы, расположение элементов на странице и создавать адаптивный дизайн под разные устройства.

  • Отделяет представление (оформление) от структуры HTML-документа.
  • Позволяет переиспользовать общие стили на множестве страниц.
  • Поддерживает гибкую систему селекторов и каскадность правил.

Пример исходного кода

<!doctype html>

                <style>

                    body {
                        font-family: Arial, sans-serif;
                    }
                
                </style>
